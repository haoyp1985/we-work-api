  - ✅ 发送对象选择（账号选择、群组导入）
  - ✅ 定时发送功能
  - ✅ 发送频率控制
  - ✅ 实时消息预览
  - ✅ 保存草稿和模板
  - ✅ 发送确认和进度预估
