## 🛠️ 开发模式特性

在开发环境 (`npm run dev`) 下：
- 自动使用模拟数据，无需后端API
- 默认为admin用户，拥有所有权限
- 菜单基于路由配置自动生成
- 支持热重载和实时预览

## 🏭 生产环境

在生产环境中：
- 将从后端API `/auth/menus` 获取菜单权限
- 根据用户真实角色过滤菜单
- 需要有效的JWT Token认证

## 🎨 用户角色权限

监控中心支持的角色：
- `admin` - 超级管理员，全部权限
- `tenant_admin` - 租户管理员，可查看监控规则
- `monitor_viewer` - 监控查看者，只读权限

## 🔧 自定义配置

如需修改菜单：
1. 编辑 `src/router/routes.ts` 中的 `asyncRoutes`
2. 修改图标：更新路由meta中的icon字段
3. 调整权限：修改路由meta中的roles数组
4. 添加新页面：在children数组中添加新路由
